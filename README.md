##  Overview

<code>❯ This is website which is intended to help user's manage their card collections.</code>

---

##  Features

<code>❯ The user is able to view all types of card, click on them to view all cards of that type, then they can use filters and searching to find their desired cards
in our diverse database.Then they can click on them fill out partiulards about their own card, and store it in their cards. 
The user's are able to click on cards to view more information about them. In their cards section they can add more cards, remove cards and see more details aboit their cards.
In the profile section user's can change thei details like, password, username and email.</code>

##  Getting Started

###  Prerequisites

Before getting started with WebDesignCoursework, ensure your runtime environment meets the following requirements:

- **Programming Language:** JavaScript
- **Package Manager:** Npm


###  Installation

Install WebDesignCoursework using one of the following methods:

**Build from source:**

1. Clone the WebDesignCoursework repository:
```sh
❯ git clone ../WebDesignCoursework
```

2. Navigate to the project directory:
```sh
❯ cd WebDesignCoursework
```

3. Install the project dependencies:


**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm install
```




###  Usage
Run WebDesignCoursework using the following command:
**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm start
```
Setup the database using the following command:
**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm run setup-db
```

###  Testing
Run the test suite using the following command:
**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm test
```
